📖 Glossaire rapide
GEH
Groupe d'Exposition Homogène – ensemble de travailleurs ayant des expositions similaires
VLEP
Valeur Limite d'Exposition Professionnelle – seuil réglementaire (8h) ou court terme (CT)
Co-prélèvement
Plusieurs agents chimiques analysés sur le même support physique (même pompe, même débit) — badge 🔵 Bleu
Fusion
Plusieurs prélèvements distincts regroupés pour un même opérateur (supports différents) — badge 🟠 Orange
Support
Milieu de prélèvement : tube charbon actif, tube gel de silice, filtre, etc.

Définir les GEH
Prépa → GEH

Ajoutez chaque Groupe d'Exposition Homogène avec + Ajouter un GEH.

Pour chaque GEH, renseignez : nom et éventuellement le nombre de personnes concernées.

👥 GEH
GEH 1 – Peintres
GEH 2 – Soudeurs
GEH 3 – Manutentionnaires
+ Ajouter un GEH
⚠️
Définissez d'abord TOUS les GEH avant de passer aux agents chimiques. L'ordre ici détermine la numérotation finale.
3
Sélectionner les agents chimiques
Prépa → Agents

Utilisez la barre de recherche pour trouver chaque agent chimique dans la base de données intégrée. Appuyez sur l'agent pour l'ajouter à la mission.

🧪 Agents chimiques
🔍 Rechercher un agent...
Résultats :
Toluène · CAS 108-88-3
Xylènes · CAS 1330-20-7
Acétone · CAS 67-64-1

Chaque agent de la base contient automatiquement :

  • VLEP 8h et CT
  • Type de support requis
  • Débits min/max
  • Codes analytiques (SCOLA)
  • Laboratoire sous-traitant
4
Affecter les agents aux GEH
Prépa → Affectations

Pour chaque combinaison agent + GEH, cochez le type de mesure à réaliser :

TypeDescriptionSous-prélèvements auto
8h RéglementaireValeur limite légale contraignante3 prélèvements
8h Non-réglementaireValeur admise / indicative1 prélèvement
CTCourt terme (15 min)1 prélèvement
💡
Co-prélèvement intelligent : si plusieurs agents d'un même GEH partagent le même type de support et le même code prétraitement, l'appli propose de les regrouper sur un seul support physique.
5
Valider la mission
Prépa → Valider

Appuyez sur ✓ Valider la mission. L'appli génère automatiquement l'ensemble des prélèvements à réaliser selon vos affectations.

✅
La mission passe en statut "Terrain" et apparaît dans la liste des missions terrain. Elle est prête à être utilisée sur site.
2
Phase terrain (sur site)
6
Ouvrir la mission sur le terrain
🏠 Accueil → Terrain

Depuis l'accueil, appuyez sur Terrain puis sélectionnez la mission dans la liste.

Codes couleur des prélèvements dans la liste :

Gris — Non démarré
Le prélèvement n'a pas encore été lancé. Aucune donnée saisie.
Orange — Fusion (en cours ou en attente)
Plusieurs prélèvements distincts sont regroupés pour un même opérateur. Chaque agent a son propre support physique. Le badge orange indique que ces prélèvements ont été fusionnés manuellement.
Bleu — Co-prélèvement
Plusieurs agents chimiques partagent le même support physique (même pompe, même débit). Le formulaire est simplifié : les débits et la pompe ne se saisissent qu'une seule fois pour tous les agents du groupe.
Vert — Terminé
Toutes les données ont été saisies et le prélèvement a été coché "Terminé".
💡
Différence clé : le bleu (co-prélèvement) signifie qu'un seul support est utilisé pour plusieurs agents. Le orange (fusion) signifie que plusieurs supports distincts sont regroupés pour un seul opérateur.
7
Démarrer un prélèvement
Terrain → Prélèvement

Appuyez sur un prélèvement dans la liste pour l'ouvrir. Saisissez le débit initial et le numéro de pompe avant de démarrer.

Appuyez sur ▶ Démarrer pour lancer le chronomètre.

⏱
Le chronomètre continue même si vous sortez de l'écran ou mettez l'appli en arrière-plan. Vous pouvez naviguer librement entre les prélèvements.
8
Saisir les données de prélèvement
Terrain → Formulaire

Pour chaque prélèvement (ou sous-prélèvement J1/J2/J3 pour les 8h réglementaires), renseignez :

ChampDescriptionAlerte auto
OpérateurNom du travailleur suivi—
DateDate du prélèvement—
N° PompeIdentifiant de la pompe—
Réf. échantillonRéférence labo du support—
Débit initialEn L/min, avant prélèvement—
Débit finalEn L/min, après prélèvement🔴 si variation > 5%
🔴
Si la variation de débit dépasse 5% entre initial et final, un avertissement rouge s'affiche. Ce prélèvement sera à noter dans le rapport.
🎤
Dictée vocale : appuyez sur l'icône micro à côté d'un champ texte pour dicter la valeur à la voix, utile quand les mains sont occupées.

Quand plusieurs agents sont sur le même support physique (co-prélèvement), le formulaire est simplifié :

  • Débits saisis une seule fois
  • Une référence échantillon distincte par agent (pour le labo)
  • Badge bleu 🔵 visible dans la liste
9
Arrêter le prélèvement
Terrain → Formulaire

À la fin du prélèvement, saisissez le débit final puis appuyez sur ⬛ Arrêter.

Le chronomètre s'arrête et le temps est enregistré. Cochez "Terminé" une fois toutes les données saisies.

✅
Utilisez le bouton "J-1" pour copier automatiquement les données du jour précédent (opérateur, numéro de pompe) sur les prélèvements multi-jours.
9b
Saisir les conditions ambiantes
Terrain → Conditions

Depuis la vue mission, accédez à 🌡 Conditions pour enregistrer par jour de prélèvement :

🌡️ Température (°C) · Initial / Final
📊 Pression (hPa) · Initial / Final
💧 Humidité (%) · Initial / Final

Depuis la vue mission terrain, appuyez sur 📋 Échantillons puis sur ⬇ Export Excel.

Le fichier généré contient :

Feuille ExcelContenu
PrélèvementsTous les prélèvements avec opérateurs, débits, références, durées
ConditionsT°, pression, humidité par jour
ÉchantillonsListe analytique avec codes SCOLA, supports, prétraitements
Blancs terrainListe des blancs avec références
📁
Le fichier Excel est téléchargé directement sur votre appareil. Pensez à le transférer vers votre réseau ou à l'envoyer par mail dès que vous avez une connexion.
11
Exporter le rapport d'activité (Word)
Terrain → Échantillons → Export Word

Le rapport Word est un document formaté qui résume l'ensemble de la mission terrain. Il est destiné à être intégré dans votre rapport final ou transmis à votre responsable.

Comment générer le rapport :

1
Depuis la vue mission terrain, appuyez sur 📋 Échantillons
2
Appuyez sur ⬇ Export Word — le fichier se génère automatiquement
3
Le fichier .docx est téléchargé sur votre appareil, prêt à être ouvert dans Word ou envoyé par mail

Contenu du rapport Word :

SectionContenu
En-tête missionClient, site, préleveur, débitmètre, contrat, dates
Tableau prélèvementsListe complète par GEH avec opérateur, agent, type, pompe, débits, durée, référence
Conditions ambiantesTempérature, pression, humidité par jour de prélèvement
Blancs terrainRéférences et agents associés
AlertesPrélèvements avec variation de débit > 5% signalés
📝
Le fichier Word est entièrement modifiable. Vous pouvez l'ouvrir dans Microsoft Word ou Google Docs pour ajouter des observations, corriger des données ou mettre en page avant envoi.
⚠️
Vérifiez que tous les prélèvements sont cochés "Terminé" et que les conditions ambiantes sont renseignées avant de générer le rapport Word — ces données s'y retrouvent directement.
12
Ajouter des blancs terrain
Terrain → Échantillons → + Blanc terrain

Un blanc terrain est un support non exposé utilisé comme témoin. Pour l'ajouter :

Appuyez sur + Blanc terrain, sélectionnez l'agent chimique concerné, saisissez la référence de l'échantillon et la date.

Les blancs apparaissent dans la liste des échantillons et sont intégrés à l'export Excel et Word.

Conseils pratiques & astuces
📱 Installation de l'appli (PWA)

Pour installer l'application sur votre téléphone sans passer par un store :

Sur Android (Chrome)
  1. Ouvrir l'URL dans Chrome
  2. Menu ⋮ → "Ajouter à l'écran d'accueil"
  3. Confirmer l'installation
Sur iPhone (Safari)
  1. Ouvrir l'URL dans Safari
  2. Bouton partage □↑ → "Sur l'écran d'accueil"
  3. Confirmer l'installation
🔄 Mise à jour de l'application

La mise à jour est automatique : à chaque lancement, l'application vérifie si une nouvelle version est disponible et se recharge d'elle-même en quelques secondes.

💡
Si l'appli ne semble pas à jour après un lancement, fermez-la complètement (swipe away sur mobile) puis rouvrez-la. La mise à jour se déclenche alors automatiquement.
✅ Bonnes pratiques
  • Préparer la mission la veille
  • Exporter l'Excel en fin de journée
  • Saisir les conditions ambiantes matin et soir
  • Vérifier la variation de débit avant de décrocher
❌ À éviter
  • Vider le cache du navigateur en cours de mission
  • Utiliser le mode navigation privée
  • Fermer l'appli sans avoir exporté
  • Modifier une mission validée sans précaution
⚡ Import depuis tableau de devis (Prépa automatique)

Pour les missions importantes, vous pouvez importer directement depuis un tableau Excel de devis. L'appli reconnaît plusieurs formats et génère automatiquement les GEH, agents et affectations.
